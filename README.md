# 2048
2048 app with C#

# Logic de base du jeu :

Règle du jeu :

- A la base du jeux, 2 nombres se créent sur la grille aléatoirement entre 2 et 4.
- A chaques actions, 1 nombre aléatoire entre 2 et 4 va apparaitre sur la grille.
- Le jeu va permettre en entrée, les flèche directionelles.
- La direction entrée va tasser les nombres si ils sont égaux et faire le double sur la zone tassée.
- Un score est affiché et permet de quantifier les tassement.

Grille :

- La grille est composée de 4 lignes et de 4 colonnes.

Tuiles :

- Les tuiles sont les emplacement sur la grille qui permettent de placer des nombres.

Mouvements :

- Les mouvements autorisé sont les flèches directionelles.

Score :

- Le score s'additione lors de tassement. La valeur du nouveau chiffre s'ajoute au score.
